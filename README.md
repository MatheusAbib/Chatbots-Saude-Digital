## 📖 Sobre o Projeto

Este projeto consiste em um **artigo acadêmico digital** que realiza uma análise comparativa estruturada de três das principais plataformas de chatbots utilizadas na área da saúde. O conteúdo aborda aspectos como precisão diagnóstica, conformidade com a LGPD, satisfação dos usuários, acessibilidade técnica e compreensão de linguagem natural.

### 🎯 Objetivos da Pesquisa

- Comparar tecnicamente as plataformas Ada Health, Molly (Sensely) e Symptomate
- Avaliar a conformidade com a Lei Geral de Proteção de Dados (LGPD)
- Analisar a satisfação dos usuários através de avaliações em lojas de aplicativos
- Mensurar a acessibilidade técnica utilizando ferramentas como Google Lighthouse
- Testar a compreensão de linguagem natural através do método "Golden Set"
